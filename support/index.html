Support

The fastest way to get help is to send a clear recording of the issue. Screen Flow Pro is built for that.

1) Create a Support Report (Pro)

In the app, choose Pro: Support Report or Pro: Bug Report (Transcript + Markers).

  • Add timeline markers at important moments
  • Export and share the generated files to your support channel
  • Bug Report mode includes a .VTT transcript + a marker summary

2) Include this info

  • Device model + iOS/iPadOS version
  • What you expected vs. what happened
  • Whether the issue reproduces consistently
  • If it involves sharing: which target app (Messages, Mail, Drive, etc.)

Email: Aleandri-Creative@outlook.com

Common fixes

  • Recording failed: Make sure Screen Recording is enabled in Control Center. Restart the device if the broadcast picker gets stuck.
  • Unexpected notifications: Turn on Do Not Disturb before recording.
  • Restore Pro: App → Settings → Restore Purchases.
